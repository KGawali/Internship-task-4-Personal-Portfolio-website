Tic Tac Toe Game


As part of my portfolio project, I developed a classic Tic Tac Toe game using HTML, CSS, and JavaScript. This timeless game offers a simple yet enjoyable experience for players of all ages. With its straightforward rules and engaging gameplay, this project showcases my skills in frontend development and interactive web design.
